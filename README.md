# DoorDash Dashboard

React + Vite + TypeScript prototype.

## Structure

- `src/components/ui` — reusable primitives (buttons, inputs, cards)
- `src/components/layout` — shell/nav/page-frame components
- `src/components/charts` — chart/visualization components
- `src/features` — feature-specific modules (grouped by domain)
- `src/pages` — top-level routed views
- `src/hooks` — shared custom hooks
- `src/lib` — utilities, helpers, API clients
- `src/data` — mock/static data for the prototype
- `src/types` — shared TypeScript types
- `src/styles` — global styles
- `src/assets` — images, icons

## Getting started

```bash
npm install
npm run dev
```
